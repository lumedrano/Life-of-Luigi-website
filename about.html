Projects and Experiences

Listed are some of the projects I have been a part of that have contributed to my skills.

Blue Origin Internship AI Model Project

This project was centered around creating an AI model for Blue Origin Aerospace. I was able to design this chatbot from scratch using TensorFlow and PyTorch along with other various frameworks without using plug ins. The result was a fully operational chatbot that was released company wide increasing productivity by 80%!

ECE319K Final Embedded Systems Lab

For our final project of the semester, my partner and I implemented DAC, ADC, Serial COM, C Programming, and more to implement our video game.

Space Invaders using C Programming

Upon finishing our circuit board, we moved onto programming the movements in C to achieve a bit mapping front-end with a multi-functional IO backend.

Block Detection Algorithm (TensorFlowObjecDetection)

In FTC Robotics the originally provided TF code for object detection was not the best. To do better, I created an algorithm that would quick sort a single frame, take the middle pixels, and find the darkest area in comparison to its distance towards the half point. This would allow us to have quick and efficient sensing in matches

Robot Design and Programming Competition

For the beginning of the season, I designed a robot with my team but led the software side to prepare an autonomous and teleoperational script to run and accomplish over 80% of tasks in one week after game reveal.

3 day built robot for Regionals

As my team went to the half way point of the season we decided to completely rebuild our bot to have more capabilities than before. We tore down everything from its lifting mechanism to its electronics. From there I planned as captain to designate 3 days, each for one section of the bot. First day would be base, second would be lifting mechanism, and last would be electronics. Doing so we had a 60% increase in point scoring with a ticket to state!
